     通用名称服务



1. 名称的级别

        本系统支持的名称可以分为三个级别。举例说明：com是一级名称。qq.com是二级名称。abc.qq.com是三级名称。

        在dapp页面，点击一级名称，在下方会显示其下的二级名称，点击二级名称，在下方会显示其下的三级名称。

2. 永久拥有

        用户在本系统创建或者购买的名称由用户永久拥有。名称可以使用中文、英文、数字、符号以及世界其它语言。

3. 用户分享系统收入

        拥有一级名称或者二级名称的用户，可以和本系统分享收入。

        当用户创建二级名称的时候，拥有这个二级名称对应的一级名称的用户，可以获得本系统收入的38%。

        例如：A用户拥有一级名称com，当B用户创建二级名称qq.com的时候，系统会向B用户收取一定费用，此时A用户可以获得系统收入的38%。

        当用户创建三级名称的时候，拥有这个三级名称对应的二级名称的用户，可以获得本系统收入的32%，拥有这个三级名称对应的一级名称的用户，可以获得本系统收入的38%。

        例如：A用户拥有一级名称com，B用户拥有二级名称qq.com，当C用户创建三级名称ccc.qq.com的时候，B用户可以获得系统收入的32%，A用户可以获得系统收入的38%。

4. 交易功能

        名称的拥有者可以在系统页面上直接将名称挂单出售，别的用户可以直接购买。

        用户如果在名称市场中看中某个名称，可以对这个名称挂上采购订单。

        如果别的用户也看中了这个名称，可以挂一个价格更高的采购订单，相当于竞拍模式。

        如果名称的拥有者觉得采购订单的价格可以接受，那么可以直接将名称卖给采购订单。

        用户在下采购订单的时候，是直接按照采购价格向本系统进行了转账交易。当用户撤销采购订单的时候，会收到全额退款。当用户的采购订单被别的用户更高价格的采购订单取代时，也会收到全额退款。

5. 用户分享交易手续费

        在本系统交易名称的时候，系统收取成交金额的5%作为手续费。

        当交易二级名称的时候，手续费中的38%，会分享给对应一级名称的拥有者。

        当交易三级名称的时候，手续费中的32%，会分享给对应二级名称的拥有者。手续费中的38%，会分享给对应一级名称的拥有者。

6. 解析功能与用户分享收入

        本系统所有级别的名称都可以进行解析，解析的目标或者种类支持btc/eth/ltc/etc/ipv4/ipv6等等常见的区块链地址以及其它内容。

        每个名称都可以进行一个或者多个解析。创建解析之后，用户还可以删除解析。

        用户为某个名称创建或者修改一条解析记录，系统收取1.2EOS的费用。

        如果是二级名称，其中的0.12EOS，会分享给对应一级名称的拥有者。

        如果是三级名称，其中的0.08EOS，会分享给对应二级名称的拥有者。其中的0.12EOS，会分享给对应一级名称的拥有者。

     Common Name Service



1. Level of name

        The names supported by this system can be divided into three levels. For example: com is a first-level name. qq.com is a second-level name. abc.qq.com is a third-level name.

        On the dapp page, click on the first-level name, and the second-level name under it will be displayed below, and click on the second-level name, and the third-level name under it will be displayed below.

2. Permanent possession

        The name created or purchased by the user in this system is permanently owned by the user. The name can use Chinese, English, numbers, symbols and other languages of the world.

3. Users share the system income

        Users with first-level or second-level names can share income with the system.

        When a user creates a second-level name, the user who owns the first-level name corresponding to the second-level name can obtain 38% of the revenue of the system.

        For example: user A has a first-level name com, when user B creates a second-level name qq.com, the system will charge user B a certain fee, and user A can get 38% of the system's revenue at this time.

        When a user creates a third-level name, the user with the second-level name corresponding to the third-level name can get 32% of the system's income, and the user with the first-level name corresponding to the third-level name can get 38% of the system's income.

        For example: user A has the first-level name com, user B has the second-level name qq.com, when user C creates the third-level name ccc.qq.com, user B can get 32% of the system revenue, and user A can get 38% of the system revenue.

4. Transaction function

        The owner of the name can place an order to sell the name directly on the system page, and other users can buy it directly.

        If a user takes a fancy to a name in the name market, he can place a purchase order for the name.

        If other users also like this name, they can place a purchase order with a higher price, which is equivalent to the auction mode.

        If the owner of the name finds the purchase order price acceptable, the name can be sold directly to the purchase order.

        When the user places a purchase order, he directly transfers money to the system according to the purchase price. When the user cancels the purchase order, they will receive a full refund. A user will also receive a full refund when their purchase order is replaced by a higher priced purchase order from another user.

5. Users share transaction fees

        When trading names in this system, the system charges 5% of the transaction amount as a handling fee.

        When trading a second-level name, 38% of the handling fee will be shared with the owner of the corresponding first-level name.

        When trading a third-level name, 32% of the handling fee will be shared with the owner of the corresponding second-level name. 38% of the handling fee will be shared with the owner of the corresponding first-level name.

6. Resolve function and user share revenue

        Names at all levels of this system can be resolved, and the resolved target or type supports btc/eth/ltc/etc/ipv4/ipv6 and other common blockchain addresses and other content.

        Each name can add one or more resolution records. After a resolution record is created, the user can also delete it.

        When a user creates or modifies a resolution record for a certain name, the system charges a fee of 1.2 EOS.

        If it is a second-level name, 0.12 EOS will be shared with the owner of the corresponding first-level name.

        If it is a third-level name, 0.08 EOS will be shared with the owner of the corresponding second-level name. 0.12 EOS will be shared with the owner of the corresponding first-level name.
